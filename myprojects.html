My Projects

1 - This website

This web site is one of many projects developed for My Higher Diploma in Computing.
It was coded according to the best practices of HTML, CSS and Java Script (including jQuery library).

Guideline of Design

Gestalt Laws: The following laws are applied to my prototypes.

a) Similarity and Proximity: Similar items are grouped in a pattern together to seen as belonging to the same group of items. The social media icons are grouped in a sequence of horizontal squares (in a row) with the same size. They are also set to same proximity (= same distance) of each other. See the annotation (1) on desktop prototype.

b) Similarity and Proximity: Another example of similarity and proximity is the set of links displayed on both sides. They are a sequence of rectangles increased one by one by the same amount of pixels. They are easily perceived as a group. See the picture below and the annotation (2) on desktop prototype.

c) Symmetry in design can creates a sense of cohesion, balance and stability that is pleasant to the viewer. All pages of this website are symmetry.

Nielsen’s heuristics:

a) The system has a simple and efficient emergency exit to “fix” user error. All pages (except home page) has a home page button.

b) The website is designed to prevent errors. The essential information from the contact form is set to ‘required’. The form informs which fields are optional. This prevents users to forget to fill essential information.

Automatic processing:

The prototypes were designed not to require users to pay attention, or consciously think on it. It is designed to be easy to navigate like many others websites across the internet which users use to visit. They do not need to learn how to navigate on it.
